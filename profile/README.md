## Intro
세감은 세종대학교 구성원들이 여러 가지 학교의 웹서비스들을 이용할 때 느끼는 치명적인 UX적인 불편함을 Use Case에 적절한 UI 및 직관적인 UX로 해결하는 앱 서비스입니다.
